# Animated Skills Bar in HTML & CSS | Progress Bar

Um projeto simples de **barra de habilidades animada (progress bar)** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar **barras de progresso animadas**, ideais para mostrar habilidades em portfólios.

## 🛠️ Tecnologias

- HTML
- CSS (animation, transition)

## 🎯 Funcionalidades

- Animação nas barras de progresso
- Design simples e moderno
- Fácil de personalizar

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
